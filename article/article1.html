ChatGPTに聞く自作HPの作り方

自分だけのホームページを持つことは、情報発信やポートフォリオ作成において非常に重要です。本記事では、ChatGPTを活用して自作のホームページを作成する方法を詳しく解説します。

準備するもの

  • 基本的なHTML/CSSの知識
  • テキストエディタ（例: VSCode）
  • GitHubアカウント

基本的なHTML構造

まず、ホームページの基本的なHTML構造を理解しましょう。以下はシンプルなHTMLの例です。

                
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Simple Blog</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header>
        <h1>My Simple Blog</h1>
    </header>
    
    <main>
        <article>
            <h2>ブログの最初の記事</h2>
            <p>ここにブログの本文が入ります。これはサンプルテキストです。</p>
        </article>
    </main>

    <footer>
        <p>© 2024 My Simple Blog</p>
    </footer>
</body>
</html>

GitHubにアップロード

次に、作成したHTMLファイルをGitHubにアップロードして公開します。GitHub Pagesを使えば、簡単に自作のHPを公開することができます。

デザインを整える

CSSを使って、ページのデザインを整えます。例えば、以下のようにスタイルを定義できます。

                
/* styles.css */
body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
    margin: 0;
    padding: 0;
    background-color: #f4f4f4;
}

header {
    background: #333;
    color: #fff;
    padding: 10px 0;
    text-align: center;
}

main {
    padding: 20px;
}

おわりに

以上が、ChatGPTを使って自作ホームページを作成する基本的な流れです。この記事を参考に、ぜひ自分だけのホームページを作成してみてください。
